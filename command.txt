1. git init: to initialise git repository.
2. git add .: to track the file. 
3. git commit -m "desizer commint": to stage the file.
4. git checkout -b branch name: To checkout and switch the branch. -b copies all the comminets before switching the branch.
5. git branch -d name_of_branch: To delete branch.
6. git status
7. git rm --cached <file_name>: to remove tracked file.
8. git log: to check hashed commit
